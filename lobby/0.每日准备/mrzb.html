照相馆食堂前厅准备工作

岗前准备：

  1. 更换工作服
  2. 洗手
  3. 开启设备：店内照明、音乐、电视、收银系统、饮水机
  4. 将展示牌放到店外

卫生打扫

  1. 桌、椅
  2. 门、窗
  3. 扫地
  4. 拖地

物料准备（早班和晚班前都要进行该项工作）

  1. 小食甜品：芥末章鱼、土豆泥沙拉、抹茶奶酪、卷心菜丝
  2. 酒水饮料：波子汽水、可乐、雪碧、乌龙茶、朝日啤酒、梅子酒
  3. 酱料：沙拉酱、猪排酱、番茄酱、抹茶粉
  4. 茶水自助区：餐盘、水杯摆放整齐，水壶接满水
  5. 外卖：打包袋、一次性餐具数量充足
  6. 每桌筷子不少于12双，餐巾纸数量充足
