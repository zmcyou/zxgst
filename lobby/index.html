照相馆食堂前厅工作流程

准备：

  1. 开启设备：店内照明、音乐、电视、收银系统、饮水机
  2. 更换工作服
  3. 洗手

基础清洁：

  1. 用对应抹布清洁桌子、椅子
  2. 用对应的抹布清洁店门，窗每周二清洁
  3. 清扫地面
  4. 拖地

物料准备（早班和晚班前都要进行该项工作）：

  1. 小食甜品：芥末章鱼、土豆泥沙拉、抹茶奶酪、卷心菜丝
  2. 酒水饮料：波子汽水、可乐、雪碧、乌龙茶、朝日啤酒、梅子酒
  3. 酱料：沙拉酱、猪排酱、番茄酱、抹茶粉
  4. 茶水自助区：餐盘、水杯摆放整齐，水壶接满水
  5. 外卖：打包袋、一次性餐具数量充足
  6. 每桌筷子不少于12双，餐巾纸数量充足

服务客人：

  1. 注意到客人即将进店：面带微笑，帮助客人开门，礼貌问候
  2. 引导顾客入座，提示顾客二维码点餐，茶水自取
  3. 上餐：检查餐品质量，确定桌号，用托盘上餐。特别商品提醒顾客注意产品冷热
  4. 客人用餐后离开，礼貌送客
  5. 收餐：餐车收餐，使用消毒液和干湿抹布清洁桌椅，并留意地面是否有垃圾

其他工作：

  1. 将洗好的餐具：筷子、勺子、小盘分类放置
  2. 留意是否需要补充筷子和餐巾纸

门店打烊

  1. 倒掉水壶中剩余的水
  2. 把垃圾放入垃圾站
  3. 将店外展示牌、清洁物品放入店内
  4. 关闭设备：店内照明、音乐、电视、收银系统、饮水机
